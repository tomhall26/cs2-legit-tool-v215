## What it does

CS2 Legit Tool is aimed at Counter-Strike 2 players who want optional aim assistance, on-screen information overlays, and movement automation in a single Windows application. It connects to the game client in real time and adds enhancements without altering any game files. In v2.1.5, the tool brings tighter detection-avoidance handling and better resilience with recent CS2 updates.

The experience is intentionally kept simple, with very little setup required. Individual modules can be switched on or off at any time, including aim assist, ESP overlay, bunnyhop automation, and trigger response behavior. Ongoing development continues to prioritize compatibility with current CS2 builds while using implementation methods intended to reduce VAC detection risk.

## Included Features

- **ESP Visual Overlay** - Shows enemy and player positions, health values, weapon details, and distance markers in a customizable on-screen overlay
- **Aimbot Module** - Supplies configurable aim assistance with adjustable smoothness, FOV constraints, and bone selection options
- **Bunnyhop Automation** - Handles jump timing automatically for steadier movement, with a configurable activation key
- **Triggerbot System** - Fires automatically when the crosshair reaches valid targets, with adjustable delay and hitbox filtering
- **Stealth Operation** - Uses memory access patterns built to reduce detectable signatures during play
- **One-Click Setup** - Ships as a single executable with no external dependencies or complicated installation steps
- **Real-Time Configuration** - Lets you enable or disable modules in-game through customizable hotkeys without restarting
